## ローカル HTTPS の証明書作成（mkcert）

ローカルで `https://localhost` を使うための自己署名証明書を、mkcert で簡単に作成できます。

1. mkcert のインストール（macOS）

```
brew install mkcert
```

2. ルート CA をインストール

```
mkcert -install
```

3. `localhost` 用の証明書を作成（リポジトリ直下で実行）

```
mkcert localhost
```

上記で `localhost.pem` と `localhost-key.pem` が作成されます（本リポジトリではこれらのファイル名を想定）。
